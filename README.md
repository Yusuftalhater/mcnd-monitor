# MCND Monitor

Fare tıklama sırası ile `mcnd.service`'i kontrol eden Debian servisi.

**Sıralar:**
- Sağ üst → Sol üst → Alt orta → **mcnd.service durdurulur**
- Sol üst → Sağ üst → Alt orta → **mcnd.service başlatılır**

## Tek Komutla Kurulum

```bash
git clone https://github.com/Yusuftalhater/mcnd-monitor.git && cd mcnd-monitor && sudo chmod +x ./install.sh && sudo ./install.sh
```
